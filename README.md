# RealidadAlterada

## 📌 Project Overview
RealidadAlterada is a Python-based web application prototype exploring interactive and dynamic interfaces with backend logic and frontend templates.

This project includes routing, template rendering, and static assets to illustrate a full stack demo web application.

---

## 🧠 Description
RealidadAlterada demonstrates backend-frontend integration using Python together with HTML, CSS, and JavaScript templates. The goal is to showcase how backend logic can serve dynamic content and manage interactive views. This serves as a basis for further enhancement with features like object interaction or augmented logic.

## 🧪 Installation & Setup

1. **Clone the repository**
```bash
git clone https://github.com/Zann181/RealidadAlterada.git
cd RealidadAlterada
````

2. **Create and activate a Python virtual environment**

```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
```

3. **Install dependencies**

```bash
pip install -r requirements.txt
```

4. **(If using Django) Apply database migrations**

```bash
python manage.py migrate
```

5. **Run the development server**

```bash
python manage.py runserver
```

6. **Visit in your browser**

```
http://localhost:8000
```
